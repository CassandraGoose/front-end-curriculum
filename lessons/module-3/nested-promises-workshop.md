### Setup

Clone down the [nested-promises](https://github.com/turingschool-examples/nested-promises) repo and start it up.

**CoWorkers Api**
```bash
git clone https://github.com/turingschool-examples/nested-promises.git
cd nested-promises
npm i
npm start
```
